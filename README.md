# Maryam Fashion Arts (maryanfashionart.pk)

High-end, production-ready luxury fashion e-commerce platform built with Next.js App Router, TypeScript, Tailwind CSS, Node.js, and PostgreSQL.

## Features
- **Luxury Aesthetic:** Black, gold, and soft beige color palette with glassmorphism and WCAG AA accessibility.
- **Maryam Fashion Assistant:** Multilingual AI chatbot supporting English, Urdu, and Hinglish with WhatsApp escalation.
- **Local & Global Payments:** COD, JazzCash, Easypaisa, Bank Transfer, and Stripe.
- **Admin & Customer Dashboards:** Comprehensive inventory, order tracking, and profile control.
